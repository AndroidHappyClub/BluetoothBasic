# 安卓蓝牙Demo

## 权限设置

需要开启**蓝牙权限**和**定位权限**

开启定位权限是由于**Android 10.+**以上的要求

## 操作提示

应用Demo主要提供以下几种功能：

  使得自己的设备能够被查看到
